## Installment

```bash
$ npm i autoscrollto --save-dev
```

## Usage

```javascript
import { AutoScrollTo } from 'autoscrollto';
const autoScroll = new AutoScrollTo();
setTimeout(() => {
  autoScroll.scrollTo();
}, 1000);
```
